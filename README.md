# 📖 About

**BookMyDoctor** is a full-stack Doctor Appointment Booking System developed using the **MERN Stack**.

The application allows users to browse doctors, book appointments, manage bookings, while administrators can manage doctors, appointments, and monitor platform statistics.

---

# ✨ Features

## 👨‍⚕️ User Features

- User Signup/Login
- JWT Authentication
- Browse Doctors
- Doctor Details Page
- Book Appointment
- My Appointments
- Cancel Appointment
- Protected Routes

---

- Dashboard Analytics
